# GdzieGraja
GdzieGrają to minimalistyczna, statyczna aplikacja webowa agregująca koncerty jazzowe we Wrocławiu. Sercem projektu jest asynchroniczny crawler (Python + Crawl4AI), który raz na dobę przeczesuje strony klubów oraz bileterii, filtruje dane przez autorską bazę znanych artystów, a następnie kompiluje i wdraża frontend (Vite + React) na GitHub Pages.
